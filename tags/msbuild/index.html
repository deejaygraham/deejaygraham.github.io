MsBuild Target Order

15 February 2014

MsBuild version 4.0 introduced two new Target attributes: *BeforeTargets* and *AfterTargets*. Prior to v4.0 if you wanted to run some tasks before or after a target you had to explicitly build them into the dependency chain using the *DependsOnTargets* attribute. For example, if you wanted to do something like print a set of configuration messages before running a build, you had to add that target to the list of build dependencies. more
